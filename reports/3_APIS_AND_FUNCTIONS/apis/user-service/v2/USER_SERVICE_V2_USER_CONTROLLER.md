# User Service API: User Controller (v2)

**Base Path:** `/api/v2/users`

This is the controller within the `user-service` that handles user-related requests for API version v2.

| Method | Path | Business Function                                | Java Method Name | Notes                                                                    |
|--------|------|--------------------------------------------------|------------------|--------------------------------------------------------------------------|
| `POST` | `/`  | Get users with enhanced filtering and sorting. | `getUsers`       | This is the backend implementation for the corresponding endpoint in the API gateway. |
