# Communication Service API Documentation

The Communication Service provides endpoints for real-time messaging, interactive proposals, and notifications.

## Key Features

- Real-time messaging through WebSocket connections
- Conversation management and message history
- Interactive proposals for coordination (postpone requests, confirmations)
- Push notification delivery via Firebase Cloud Messaging
- Message persistence and retrieval
