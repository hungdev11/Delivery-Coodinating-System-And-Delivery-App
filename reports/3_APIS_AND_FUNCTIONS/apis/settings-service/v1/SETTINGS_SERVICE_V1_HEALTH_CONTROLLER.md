# Settings Service API: Health Controller (v1)

**Base Path:** `/`

This controller provides health check endpoints for the `Settings_service`.

| Method | Path             | Business Function                                                       | Java Method Name |
|--------|------------------|-------------------------------------------------------------------------|------------------|
| `GET`  | `/health`        | Provides a health check for the Settings Service.                       | `health`         |
| `GET`  | `/actuator/health` | Provides a health check in a format compatible with Spring Boot Actuator. | `actuatorHealth` |
