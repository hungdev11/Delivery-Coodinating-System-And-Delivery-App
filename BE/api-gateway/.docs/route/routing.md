# Routing Routes (API Gateway - Proxy)

Base URL: `http://localhost:<port>/api/v1`

**Note:** These endpoints proxy requests to the Zone Service. See [Routing Service Routes](../../zone_service/.docs/route/routing.md) for detailed documentation.

## Endpoints

- `POST /routing/route` - Calculate route
- `POST /routing/demo-route` - Calculate demo route
- `GET /routing/osrm-status` - Get OSRM status

For request/response formats, see [Routing Service documentation](../../zone_service/.docs/route/routing.md).
